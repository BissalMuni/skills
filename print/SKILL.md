# 종이 출력 (printbox)

대화 내용을 종이로 출력한다. 화면을 오래 못 볼 때(눈 피로) 쓴다.
**한 번 읽고 버리는** 용도라 서식·디자인은 신경 쓰지 않는다 — 빠르고 가볍고 읽히기만 하면 된다.
**어느 프로젝트에서든** 동작한다 — 사용자 레벨 스킬이라 모든 대화에서 호출 가능.

## 전제 (고정값)

- 중앙 폴더: `D:/Coding/printbox/`
- 보관(누적): `D:/Coding/printbox/archive/` — 출력한 문서가 전부 여기에 쌓인다
- 인쇄 스크립트: `D:/Coding/printbox/print.ps1`
- 프린터: `Canon G1010 series`
- 출력 형식: **순수 텍스트(.txt)** — `Out-Printer` 로 인쇄 (Edge 불필요, 가벼움).
  글자 크기 조정은 없다. **목차·구조는 빈 줄(문단 띄우기)과 들여쓰기로** 표현한다.

## 입력

```text
$ARGUMENTS
```

- 인자가 없으면: **직전에 한 답변(또는 사용자가 가리키는 내용)** 을 출력 대상으로 삼는다.
- 인자가 있으면: 그 지시를 반영한다.

## Workflow

### Step 1: 출력할 내용 정하기

- 무엇을 종이로 볼지 결정한다. 보통 직전 답변 전체.
- 마크다운 문법(```` ``` ````, `**`, `#`, `|` 표 등)은 **종이용 순수 텍스트로 정리**한다.
- 제목(짧게)과 날짜를 정한다.

### Step 2: 텍스트 작성

`D:/Coding/printbox/archive/<YYYYMMDD-HHmmss>-<짧은제목>.txt` 로 UTF-8 저장한다.
글자 크기를 못 쓰므로, **대목차는 번호(`1.` `2.`)로, 나머지는 빈 줄과 들여쓰기로** 드러낸다
(`====`·`----` 같은 밑줄 구분선은 쓰지 않는다):

- **대목차**: `1.` `2.` 번호 + 위에 빈 줄
- **본문 문단**: 문단 사이마다 빈 줄 한 줄
- **하위 항목**: 들여쓰기(2~4칸) + `-`/`•`, 더 깊으면 더 들여쓴다
- **코드·명령**: 4칸 들여쓰기로 블록 구분

예시:

```text
취득세 세율 정리

1. 거래 성격 판정

    유상 / 무상 / 증여간주를 먼저 가른다.

    - 상속  → 무상
    - 매매  → 유상

2. 세율 적용

    제15조 특례를 먼저 확인한 뒤 제11조 표준세율.
```

### Step 3: 인쇄

저장한 txt 경로로 인쇄 스크립트를 실행한다 (Bash 도구 권장 — 경로 따옴표 주의):

```bash
powershell -NoProfile -ExecutionPolicy Bypass -File "D:/Coding/printbox/print.ps1" -Text "D:/Coding/printbox/archive/<파일명>.txt"
```

- `Out-Printer` 로 지정 프린터에 조용히(대화상자 없이) 인쇄한다 — 기본 프린터는 건드리지 않는다.
- 출력 결과 `PRINTED(text) -> ...` 가 보이면 성공.

### Step 4: 사용자에게 보고 (짧게)

눈이 피곤한 사용자다. **답변은 1~2줄로 짧게.**
- 예: "인쇄 보냈어요. archive/20260525-…​.txt 에 저장됨."

## 주의

- 한 번 읽고 버리는 문서다 — 서식에 시간 쓰지 말 것.
- 인쇄가 안 나오면: 프린터가 잠자기였을 수 있으니 잠시 후 큐 확인(`Get-PrintJob`), 또는 전원/연결·이름(`Get-Printer`) 확인.
- 서식이 꼭 필요한 예외에만 HTML + `-Html` 경로를 쓴다 (Edge 사용, 무거움).
